# Jiejie Resume

純前端個人履歷網站。不依賴後端；開發時以 Vite 提供本地伺服器與模組打包。

## 快速開始

```powershell
npm.cmd install
npm.cmd run dev
```

瀏覽器開啟終端機顯示的本機網址。

> 完整操作說明 → [docs/how-to-run.md](./docs/how-to-run.md)  
> **請勿直接雙擊 `index.html`**。PowerShell 建議使用 `npm.cmd`。

| 指令 | 說明 |
|------|------|
| `npm.cmd install` | 安裝依賴 |
| `npm.cmd run dev` | 開發 |
| `npm.cmd run build` | 建置 |
| `npm.cmd run preview` | 預覽建置結果 |
